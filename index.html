Learn how to implement JSON within an Oracle Database.
Configuration of JSON is straight forward, but the ability to manipulate JSON data requires an understanding of various new features in Oracle. We will demonstrate the primary features you will need, together with recommendations when to use each feature. We will also cover typical challenges that are encountered and how to overcome them.
Did you know, JSON has become an important feature for APEX developers?
In this webinar, we will provide answers to the following questions:

  • What is JSON and what is it used for?
  • How can I store JSON data in the database?
  • How can I update JSON data in the database?
  • How can I access JSON data that is stored in the database?
  • How do I retrieve the entire JSON data structure?
  • How do I retrieve just the parts of the JSON data that I need?
  • How can I convert JSON data into relational data to store in the database?
  • How can I convert relational data in the database to JSON if needed?
